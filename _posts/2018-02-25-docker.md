Compile your DockerFile:
```
docker build -t nerhello .
```

Run your project: 
```
sudo docker run -p 8887:8888 -it nerhello
```

Take a snapshot:
```
sudo docker commit <docker-id> nerhello2
```

Remove unnecessary images/containers etc.
```
sudo docker system prune
```
